# ubuntu-software-installer
bash script that installs programs in Linux Ubuntu

## programs included
- vim
- git
- zsh
- z (zsh plugin)
- oh-my-zsh
- curl
- tmux
- tree
- zip
- unzip
- p7zip-full
- gnome-tweaks
- postman
- docker
- docker-compose
- chrome
- vscode 
- dbeaver
- slack
- brave browser
- simplenote
